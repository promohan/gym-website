🏋️ PowerFit – Modern Gym Website



A modern and responsive gym website built using HTML, CSS, and JavaScript.
The project represents a professional fitness landing page with sections for programs, trainers, memberships, gallery, and contact.

This website was created by combining frontend development knowledge with AI-assisted development tools to build a visually appealing and interactive user interface.
